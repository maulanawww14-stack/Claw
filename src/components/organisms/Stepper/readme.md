# Stepper Organism

Sequential process indicator for multi-step workflows

## Usage

```tsx
import { Stepper } from '@/components/organisms';

const MyComponent = () => (
  <Stepper>
    Content
  </Stepper>
);
```

## Props

| Name | Type | Default | Description |
|------|------|---------|-------------|
| children | ReactNode | - | Component content |
