# DropdownMenu Molecule

Overlay menu for contextual actions or selection

## Usage

```tsx
import { DropdownMenu } from '@/components/molecules';

const MyComponent = () => (
  <DropdownMenu>
    Content
  </DropdownMenu>
);
```

## Props

| Name | Type | Default | Description |
|------|------|---------|-------------|
| children | ReactNode | - | Component content |
