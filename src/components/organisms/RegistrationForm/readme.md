# RegistrationForm Organism

Full user onboarding form with validation

## Usage

```tsx
import { RegistrationForm } from '@/components/organisms';

const MyComponent = () => (
  <RegistrationForm>
    Content
  </RegistrationForm>
);
```

## Props

| Name | Type | Default | Description |
|------|------|---------|-------------|
| children | ReactNode | - | Component content |
