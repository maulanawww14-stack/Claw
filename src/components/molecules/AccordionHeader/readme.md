# AccordionHeader Molecule

Toggle-able header for collapsible content sections

## Usage

```tsx
import { AccordionHeader } from '@/components/molecules';

const MyComponent = () => (
  <AccordionHeader>
    Content
  </AccordionHeader>
);
```

## Props

| Name | Type | Default | Description |
|------|------|---------|-------------|
| children | ReactNode | - | Component content |
