# LoadingWrapper Molecule

Higher-order component for handling data loading states

## Usage

```tsx
import { LoadingWrapper } from '@/components/molecules';

const MyComponent = () => (
  <LoadingWrapper>
    Content
  </LoadingWrapper>
);
```

## Props

| Name | Type | Default | Description |
|------|------|---------|-------------|
| children | ReactNode | - | Component content |
