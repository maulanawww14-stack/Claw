# CheckboxGroup Molecule

Managed group of industrial checkbox inputs

## Usage

```tsx
import { CheckboxGroup } from '@/components/molecules';

const MyComponent = () => (
  <CheckboxGroup>
    Content
  </CheckboxGroup>
);
```

## Props

| Name | Type | Default | Description |
|------|------|---------|-------------|
| children | ReactNode | - | Component content |
