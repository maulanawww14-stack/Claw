# FormField Molecule

Industrial form field wrapper combining label, input, and error messages

## Usage

```tsx
import { FormField } from '@/components/molecules';

const MyComponent = () => (
  <FormField>
    Content
  </FormField>
);
```

## Props

| Name | Type | Default | Description |
|------|------|---------|-------------|
| children | ReactNode | - | Component content |
