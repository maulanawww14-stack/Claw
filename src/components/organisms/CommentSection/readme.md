# CommentSection Organism

User engagement area with threaded replies

## Usage

```tsx
import { CommentSection } from '@/components/organisms';

const MyComponent = () => (
  <CommentSection>
    Content
  </CommentSection>
);
```

## Props

| Name | Type | Default | Description |
|------|------|---------|-------------|
| children | ReactNode | - | Component content |
