# CardFooter Molecule

Industrial card footer for metadata or buttons

## Usage

```tsx
import { CardFooter } from '@/components/molecules';

const MyComponent = () => (
  <CardFooter>
    Content
  </CardFooter>
);
```

## Props

| Name | Type | Default | Description |
|------|------|---------|-------------|
| children | ReactNode | - | Component content |
