# Breadcrumbs Molecule

Hierarchical navigation path indicator

## Usage

```tsx
import { Breadcrumbs } from '@/components/molecules';

const MyComponent = () => (
  <Breadcrumbs>
    Content
  </Breadcrumbs>
);
```

## Props

| Name | Type | Default | Description |
|------|------|---------|-------------|
| children | ReactNode | - | Component content |
