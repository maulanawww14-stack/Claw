# HeroSection Organism

Industrial hero section with title, description and CTA

## Usage

```tsx
import { HeroSection } from '@/components/organisms';

const MyComponent = () => (
  <HeroSection>
    Content
  </HeroSection>
);
```

## Props

| Name | Type | Default | Description |
|------|------|---------|-------------|
| children | ReactNode | - | Component content |
